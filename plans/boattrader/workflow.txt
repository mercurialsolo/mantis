BoatTrader Lead Extraction → PopYachts Lead Entry

URL: https://www.boattrader.com/boats/
Requires login session: no

Inputs:
  search_radius (required) (distance in miles from zip code)
  zip_code (required) (US zip code for search center)
  pop_password (required) (password for PopYachts admin authentication)

Steps:
1. Navigate to https://www.boattrader.com/boats/
2. Enter "{{zip_code}}" in the location/zip code field
3. Set search radius to "{{search_radius}}" miles
4. Set minimum price to "35000"
5. Select "Private Seller" in the seller type filter
6. Sort by "Recently Updated" newest first
7. Wait for filtered results to load
8. For each listing on the results page, open it in a new tab
9. Check if the listing has a phone number in the Description or More Details section
10. If viable (has phone number), extract: year, make, model, boat_type, asking_price, listing_url, seller_name, seller_phone
11. If not viable, skip to next listing
12. For viable leads, check for duplicates by phone number or URL
13. If not duplicate, enter lead in PopYachts admin at https://admin.popyachts.com/
14. Authenticate with Google (svsg@popsells.com / {{pop_password}})
15. Select user "Erin NesSmith" in User Management section
16. Navigate to lead entry form at https://admin.popyachts.com/Page/Account-Main-Seller-Lead-Review-External/NA/Page.html
17. Select "Outside Boat Lead" and fill all fields
18. Submit the lead and verify URL changes
19. Close listing tab, return to search results, process next listing
20. When all listings on page are done, click Next page
21. Repeat until no more pages
22. Verify: generate completion report with all processed leads
